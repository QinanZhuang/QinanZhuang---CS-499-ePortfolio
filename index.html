Project 1: Software Design and Engineering

For my Software Engineering and Design enhancement, I improved the Room page of the Travlr web application by adding several new features that provide a better user experience and demonstrate my full-stack development skills. Originally, the Room page only displayed basic information about the available rooms. I enhanced this page by creating a complete room booking system that allows users to view room details and make reservations online. Each room now includes a picture, room description, room price, and the number of rooms remaining. This gives users more information when selecting a room that meets their needs. I also implemented a booking feature that allows users to select their check-in and check-out dates, enter personal information, and submit special requests. On top of that, I added a track booking tab on every page, allows user to track their books as well, if user wants to modify booking, there will be a customer service number for them to call.

Project 2: Algorithms and Data Structures

I enhanced the Travlr project by improving search, filtering, and application logic. The application automatically calculates and displays the total cost of the stay based on the selected room and the length of the reservation. This improvement provides users with immediate pricing information and makes the booking process more convenient. In addition, I added functionality to track room availability. Users can see how many rooms are still available before making a reservation. After a booking is successfully submitted, the system generates a unique booking number that users can use to view or verify their reservation later. These enhancements improved the usability and functionality of the application while demonstrating software engineering principles such as user-centered design, form validation, data management, and application integration. Through this enhancement, I strengthened my skills in front-end development, back-end functionality, and creating a more interactive and professional web application.

Project 3: Databases

I enhanced the database functionality by creating a booking model in MongoDB, adding API routes to save and retrieve reservations, and connecting the booking form to the database. Users can submit a reservation, receive a unique booking number, and later use that number to retrieve their booking details.
